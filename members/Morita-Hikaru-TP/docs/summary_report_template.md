## 1. 何を作ったか（30秒）

<!-- ひと言で伝わるように書いてください。「○○を使って、△△するガジェットを作りました。」 -->

**ガジェット名：**超音波センサーを使って手などをかざしてもLEDやモータが使える（モータでプロペラを使って風を吹かすことが出来ました）

**ひと言で説明：**かざすものさえあればつく

**使った部品：**
| 部品名 | 何のために使うか（機能との対応） |
|:---|:---|
| 超音波センサー（HC-SR04） | 手の距離を測定し、LED点灯とモーター起動のトリガーにする |
| LED | 手をかざしたときの視覚的フィードバックとして点灯させる |
| 220Ω抵抗 | LEDの電流制限のため |
|Power Supply Module|ブレッドボードに安全で安定した電源（5V/3.3V）を供給すること|
| DCモーター | 風を送るためのファンとして使用 |
| L298N / L293D | モーターのON/OFF制御のため |
| ジャンパー線 | 各部品の接続 |
| ブレッドボード | 回路の組み立て |
