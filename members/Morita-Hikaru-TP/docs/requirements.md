## 3. 機能要件

### 3-1. このガジェットができること（必須機能）
> ※ 「〜できる」の形で箇条書き。最低3つ書いてください。

- 超音波センサーで手の距離を測定できる

- 一定距離以内に手が入るとLEDが点灯できる

- 手をかざすとモーターが回転し風を送れる
- 手を離すとLEDとモーターが自動で停止できる

## 4. 使用部品リスト

### 4-1. グループ共通部品（グループ全員で使うもの・1つ）

**共通部品：**超音波センサー

**選定理由：**汎用性が高いため。

### 4-2. 自分が追加で使う予定の部品（ELEGOOキットより）

| 部品名 | 何のために使うか（機能との対応） |
|:---|:---|
| 超音波センサー（HC-SR04） | 手の距離を測定し、LED点灯とモーター起動のトリガーにする |
| LED | 手をかざしたときの視覚的フィードバックとして点灯させる |
| 220Ω抵抗 | LEDの電流制限のため |
| DCモーター | 風を送るためのファンとして使用 |
| L298N / L293D | モーターのON/OFF制御のため |
| ジャンパー線 | 各部品の接続 |
| ブレッドボード | 回路の組み立て |

## 5. AIレビュー記録

> ※ 下記の質問をAIに投げ、回答と自分の対応を記録してください。

### Q1: 不足の確認
> 「この要件定義書をもとに基本設計を作ろうとしたとき、不足している情報や曖昧な点はどこですか？」

AIの回答（要約）：
- 超音波センサーの役割が明確でない

- 距離の閾値が未定

- モーターの種類が曖昧

- 電源方式が不明
----

対応した内容：
- 超音波センサーを中心にした機能要件へ修正

- 距離の閾値は15cm前後を想定

- DCモーターを使用することを明記

- 電源はUSB給電を基本とする
-----
-----
